# Git Workflow (Fork + Upstream)

This repo is a fork. Remotes:

- `origin` ‚Üí your fork (`NihapMrm/Echo-Music-Desktop`) ‚Äî push your changes here
- `upstream` ‚Üí original (`EchoMusicApp/Echo-Music-Desktop`) ‚Äî pull updates from here

## Pull updates from the original repo

```
git fetch upstream
git merge upstream/main
```

## Push your own changes

```
git push origin main
```
